# AI Data Annotation Project

## Overview
This project demonstrates my experience in training AI models through data annotation tasks. It includes labeled text data for sentiment analysis and intent classification.

## Tasks Performed
- Text classification
- Sentiment labeling (Positive, Negative, Neutral)
- Intent recognition (Complaint, Inquiry, Praise, Request)

## Objective
To create high-quality labeled datasets that improve machine learning model accuracy and performance.

## Tools Used
- Microsoft Excel / Google Sheets
- Manual annotation techniques
